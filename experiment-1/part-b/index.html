HTML Lists — Demonstration

Examples showing: ordered list (<ol>), unordered list (<ul>), nested lists, an ordered list inside an unordered list, and definition lists (<dl>).

1. Ordered List (<ol>)

An ordered list displays items with numbers (1., 2., 3., ...). Use <li> elements inside the <ol>.

<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>
  1. First item
  2. Second item
  3. Third item

2. Unordered List (<ul>)

Unordered lists show bullet points. Use <ul> with <li> elements.

<ul>
  <li>Apples</li>
  <li>Bananas</li>
  <li>Cherries</li>
</ul>
  • Apples
  • Bananas
  • Cherries

3. Nested Lists

You can nest lists inside list items — either same type or mixed. This helps show hierarchy.

<ul>
  <li>Fruits
    <ul>
      <li>Citrus</li>
      <li>Berries</li>
    </ul>
  </li>
  <li>Vegetables</li>
</ul>
  • Fruits
    • Citrus
    • Berries
  • Vegetables
Tip: Indent nested lists for readability in your HTML source.

4. Ordered List inside an Unordered List

You may place an ordered list inside an unordered list item (or vice versa) — useful for step groups within categories.

<ul>
  <li>Setup steps
    <ol>
      <li>Install software</li>
      <li>Configure settings</li>
    </ol>
  </li>
  <li>Notes</li>
</ul>
  • Setup steps
    1. Install software
    2. Configure settings
  • Notes

5. Definition List (<dl>)

Definition lists pair a term (<dt>) with one or more descriptions (<dd>).

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
HTML
HyperText Markup Language
CSS
Cascading Style Sheets
Definition lists are great for glossaries, FAQs, or term/description pairs.

That's it — a compact demo of HTML list types. Copy this file, open in a browser, and experiment by adding more nested items or mixing types.
